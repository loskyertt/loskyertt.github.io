2.2 zsh prompt

Code Info
%T 系统时间（时：分）
%* 系统时间（时：分：秒）
%D 系统日期（年-月-日）
%n 用户名称（即：当前登陆终端的用户的名称，和whami命令输出相同）
%B 开始到结束使用粗体打印
%b 开始到结束使用粗体打印
%U 开始到结束使用下划线打印
%u 开始到结束使用下划线打印
%d 你当前的工作目录
%~ 你目前的目录相对于～的相对路径
%M 计算机的主机名
%m 计算机的主机名（在第一个句号之前截断）
%l 你当前的tty
%F{色码} 用来设定某个颜色的开始
%f 用来设定成预设的样式， 也可以说是设定好的颜色结束
